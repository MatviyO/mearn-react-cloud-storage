0.Структура домена(У нас один домен для всех страниц) https://true-medicine.com/
	-> В корне домена лежат папки с названием оффером
		-> В каждой папке лажит 3 типа страниц
			1."p" - проклы
				-> номера страниц
					->Цена(если указано на прокле)
			2."lp" - лендинги
			3."plp" - прокла-лендинг(прокла с формой)
				->для лендингов и прокла-лендингов: номера страниц внутри которых лежат 
							->папки с ценами на этих лендингах

Я основную часть наших дополнительных библиотек вынес в отдельную папку и свёл к 3 подключаемым файлам(в корень страницы их загружать не надо).
Они сами подключают: LP Protect, Jquery(min), 2-а пуша, яндекс метрику, маску для номера и фильтр, ретаргеты с тигерок, ивенты для трекера и реквизиты в футере!Немного позже будет ещё витрина)
1.Перед <html>, до начала загрузки страницы:
	<?php $root = substr($_SERVER['DOCUMENT_ROOT'],0,-14); include($root.'/includes/pre.php'); ?>
	
2.Перед закрывающим тегом </head>:
	<?php include($root.'/includes/header.php');?>
	2.1.Если надо выключить JS:
	<?php $js=false; include($root.'/includes/header.php');?>
	
3.Перед закрывающим тегом </body>:
	<?php include($root.'/includes/footer.php');?>
	
!!!ВНИМАНИЕ!!! 
1.Если это ленд или пре-ленд тогда переделываем форму(ниже обязательные поля):
	<input type="text" name="name" required="required">
    <input type="text" name="phone" required="required">
    <input type="hidden" name="offerId">
    <input type="hidden" name="price">
	
2.При переделывании страницы убираем всё лишнее! 
Проверяем страницу на наличие(подключены) лишних скриптов! 
Особенно обращаем внимание на скрипты пушей, редиректов, подмен данных страницы! 
Я регулярно встречаю скрипты, которые могут подгрузить свою витрину(переписав нашу), любят переписывать формы: action, offerId, price! 
Так же внимательно проверяем альты у картинок!
